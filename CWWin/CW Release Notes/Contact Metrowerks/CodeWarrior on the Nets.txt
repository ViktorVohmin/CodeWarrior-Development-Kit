
CodeWarrior on the Nets
---------------------------------------------

Check the following Metrowerks CodeWarrior forums for support and updates.

Please continue to send bug reports to our support address:
support@metrowerks.com

World-wide Web Site:
 http://www.metrowerks.com/

Internet newsgroups:
   For CodeWarrior for Windows 95/NT issues, read or post questions to: 
      comp.os.ms-windows.programmer.tools.misc
   For CodeWarrior for Mac OS issues, read or post questions to: 
      comp.sys.mac.programmer.codewarrior
   For Java related issues, read or post questions to:
      comp.lang.java
   For PowerPlant related issues, read or post questions to:
      comp.sys.mac.oop.powerplant
 
Online services (have patches):
1) America OnLine
   Computing :: Development Forum :: Industry Connection :: Metrowerks
   keyword ---> Metrowerks
2) Compuserve
   Macintosh Developers Support Board
   goto MacDev :: Lib #16 -- Metrowerks

Online services (don't have patches):
1) GEnie
   PowerPC Developers Roundtable
   PPCPRO # 1440 :: Lib #8 Apple PowerPC:System 7
