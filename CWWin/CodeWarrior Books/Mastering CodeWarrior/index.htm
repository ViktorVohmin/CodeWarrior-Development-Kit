Mastering CodeWarrior for Windows 95/NT by Jim Trudeau

This book gives you the hands-on experience you need to fully understand and master the CodeWarrior development environment. Aimed at all programmers new to CodeWarrior, you'll encounter the CodeWarrior tools at many levels--the architecture, the design principles, and the nuts-and-bolts practical tips, tricks, and techniques you use to get real work done. This book covers the integrated development environment, the project manager, the source code editor, the browser, the debugger, and much more. The CD contains a preview of this work, which is under development and soon to be published by Sybex.
