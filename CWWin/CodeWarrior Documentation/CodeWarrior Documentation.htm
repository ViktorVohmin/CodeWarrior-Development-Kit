Core Documentation

IDE User Guide (Mac OS, Windows 95/NT4, Be OS)

The IDE User Guide gives an in-depth "how-to" description of the CodeWarrior Integrated Development Environment (IDE) for software development. This is the fundamental manual to the CodeWarrior tool set. Chapters cover how to manage projects, use the source code editor, the browser, compile and link code, set preferences and project options, and much more. The IDE User Guide does not discuss programming issues that are specific to an individual target. The User Guide is intended for all CodeWarrior users.

Debugger Manual (Mac OS, Windows 95/NT4, Be OS)

The CodeWarrior Debugger manual is the user guide for the CodeWarrior source-level debugger. This manual includes chapters on getting started with debugging, and a complete description of the debugger windows and menus. You learn how to use the debugger to step through and debug your source code, view and manipulate variables, to set and use breakpoints and watchpoints, to work with expressions, and to do basic troubleshooting. The Debugger Manual is intended for all CodeWarrior users.

CodeWarrior Error Reference

The Error Reference describes the error messages that can be generated by the CodeWarrior IDE compilers and linkers. If you encounter an error, you can learn about the error in this reference. The possible causes of the error are listed, and sometimes likely solutions are provided as well. The information provided can't solve your problems for you, but it should help you understand what's going on. The Error Reference is intended for all CodeWarrior users.

Language And Compiler Documentation

C/C++/Asm Language Reference

The C/C++/Asm Language Reference is a detailed description of the Metrowerks C/C++ front-end compiler. The compiler also supports assembly language, and this manual discusses those features as well. You will find information on every compiler option, #pragmas, runtime type identification, exception handling, and other features of both the C and C++ languages. This manual is intended for all C/C++ programmers.

Targeting Manuals

Targeting PlayStation

The Targeting PlayStation manual contains everything you need to know to use CodeWarrior to develop code for the SONY PlayStation platform. Shows you how to use CodeWarrior's MIPS compiler, linker, resource compiler, and debugger to create, manage, and debug a variety of PlayStation-based projects. Includes a tutorial on the CodeWarrior tools for Win32/x86 programming. This manual is intended for all PlayStation programmers.
