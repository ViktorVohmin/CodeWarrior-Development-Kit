CodeWarrior For BeBox DR2 - $149
Native Development Tools For The BeBox

Unleash the power of the BeBox! Now the same professional, rock-solid Integrated Development Environment (IDE) interface that Metrowerks has been shipping and improving for over two years is available to you with CodeWarrior for BeBox. CodeWarrior for BeBox provides a complete Be-native version of CodeWarrior that enables you to build BeBox applications right from the comfort of your BeBox computer system. The extensive toolset included in CodeWarrior for BeBox complements the tools already built-in to the BeBox computer system, heightening its development capacity beyond 64KB applications. Now you can build GUI or command line applications, shared libraries and add-on components of practically unlimited size for the DR8 release of the BeOS. All the familiar CodeWarrior features are there - BeBox compilers, unlimited linker and built-in editor, and now a development version of the native Be debugger. CodeWarrior for BeBox also puts headers and libraries, simple reference files, useful C++ classes, HTML format documentation for the IDE and compilers, and sample code all at your fingertips.
