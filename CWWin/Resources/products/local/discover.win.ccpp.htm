Discover Programming C, C++ For Windows 95/NT

The C/C++ Programming Starter Kit

Discover Programming: C/C++ for Windows 95/NT provides everything you need to learn C and C++ programming under Windows 95 and Windows NT. Discover Programming comes with both online documentation and reference books, including Learn C under Windows 95/NT and Learn C++ under Windows 95/NT by Dave Mark, Mastering CodeWarrior for Windows 95/NT by Jim Trudeau, Building Windows 95 Applications by Kevin Goodman and Windows 95: A Developer's Guide by Jeffrey Richter and Gordon Locke. The example programs from the books have been included, so you can work through them as you go. Based on the award-winning CodeWarrior software development package, Discover Programming has the learning tools to get you started and the professional features that will help you continue to develop as a programmer.
