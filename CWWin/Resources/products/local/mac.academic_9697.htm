CodeWarrior® Academic 96/97 - $79

This annual academic version of CodeWarrior® provides everything a student needs to develop Macintosh®, Power Macintosh, Windows® 95 or Windows NT non-commercial software and shareware. The award-winning, easy-to-use Integrated Development Environment (IDE) provides support for C/C++, Object Pascal and now includes Java tools as well. Online books, references and Apple® Guide files will take students step-by-step through the programming learning process with ease. One annual release covers the entire scholastic year. Proof of academic affiliation is required for purchase.

Specifications

Languages Supported

  • Java
  • C for all platforms
  • C++ for all platforms
  • Object Pascal for MacOS

Target platforms

  • 68K and Power Macintosh
  • Windows 95 and Windows NT for x86

Compilers

  • Sun Java compilers - available in the applet viewer, and as an integrated plugin in the IDE
  • C Compiler - NIST Certified ANSI C with MPW extensions
  • C++ Compiler - closely tracks the emerging ANSI/ISO C++ standard
  • Pascal Compiler - NIST Certified ANS Pascal with MPW® Pascal 3.3 compatible extensions
  • PowerPC Assembler - Apple's own assembler available as a plugin compiler
  • Final release of CFM68K compiler and linker (builds ODF, OpenDoc® parts, and PowerPlant shared libraries)
  • Balloon help compiler - included on the CD for the first time
  • Motorola's MCC and Apple's MrC are supported as plugin compilers (but sold seperately)

Features

  • Full Java support in the IDE - project manager, linker, editor
  • Class-browser for both C++ and Java
  • Metrowerks Java runtime environment (bytecode interpreter) - run applets, droplets and Java applications
  • Java bytecode disassembler - found in Metrowerks Java
  • Applet viewer - view and test applets under development
  • Drag and Drop editing in IDE's source code editor and preference panels
  • Disassembler in IDE - displays source and assembly simultaneously
  • Error management in IDE - edit errors directly in the error message window
  • Improved undo support
  • C++ templates & zero overhead exception handling
  • Multiple inheritance and RTTI support
  • Global optimization for PowerPC C/C++ compilers
  • Global optimization for Win32/x86 compilers
  • Class hierarchy optimization for Pascal
  • Pointer-based objects in Pascal - objects can now be allocated in handles or pointers
  • Procedural types in UPIs for Pascal
  • Instruction scheduling for PowerPC 601, 603, and 604 processors
  • PowerPlant application framework, with new networking classes
  • PowerPlant Constructor with view editor, menu editor, and fully integrated menu bar editor
  • PowerPlant Constructor display classes window, custom pane libraries
  • Updated support for OpenDoc development for 68K and Power Macintosh
  • MacApp® version 3.3.1 and MacApp2PPC 2.0.1 for Object Pascal
  • Microsoft Foundation Classes® version 4.0
  • MPW version 3.4.1
  • Metrowerks Standard Library (MSL) - now for both C and C++, includes source code

Debugging

  • Java source-level debugging for Macintosh and Power Macintosh - debug Java code and view both source and assembly (Java bytecodes)
  • Source-level debugger for Macintosh and Power Macintosh with support for SOM objects
  • Mac-hosted, source-level debugger for Win32/x86
  • Conditional breakpoints, data watchpoints and expression evaluation
  • Raw memory hex dump in the debugger
  • Debugger for Copland and multiprocessor Power Macintosh systems (developer release)

Other Features

  • Online Books:
    • Learn Java on the Macintosh, by Barry Boone

  • Online help and documentation (over 5,000 pages)
  • Selected Apple developer tools and third-party SDK's, tools and demos
  • Microsoft® Internet Explorer
  • Free technical support included with registration
  • 30 day money-back guarantee

Motorola® 68020 processor or higher, or PowerPC 601 processor or higher
8 Megabytes of RAM
System 7.1 or later
CD-ROM drive for installation
25 Megabytes of free hard disk space

Runs on Macintosh and Power Macintosh.
