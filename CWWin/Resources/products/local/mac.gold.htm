CodeWarrior Gold 10

The World's Best-Selling Macintosh Development Tools

CodeWarrior Gold, the world's best-selling suite of Macintosh® development tools, offers everything you need for serious, industrial-strength programming. This complete solution for development of Macintosh, Power Macintosh, Windows® 95, Windows NT, Magic Cap and BeBox software supports C/C++, Object Pascal and Java. Our award-winning, intuitive Integrated Development Environment (IDE) uses plug-in compilers that support all the languages and platforms you need, making it easy to expand an application's market without spending valuable time learning additional programming environments. With CodeWarrior Gold you also get a multitude of online books and references (over 5,000 pages) and Apple® Guide files for easy navigation through tutorials and examples. Plus, you receive 2 free updates and world-class technical support for a year when you register.

Specifications

Languages supported

  • Java
  • C for all platforms
  • C++ for all platforms except for Magic Cap
  • Object Pascal for Mac OS

Operating Systems Supported

  • Mac OS - 68k, PowerPC
  • Windows 95 - x86, Pentium®
  • Windows NT - x86, Pentium
  • BeOS - PowerPC
  • Magic Cap - 68349

Processor Families Supported

  • Motorola® 680x0
  • PowerPC 601, 603, 604
  • Intel® x86, Pentium
  • MIPS® R3000, R4200, R4600
  • Java Virtual Machine

Compilers

  • Sun Java compilers - available in the applet viewer, and as an integrated plug-in in the IDE, now including a developer release of the Metrowerks Just-In-Time (JIT) compiler
  • C Compiler - NIST Certified ANSI C with MPW® extensions
  • C++ Compiler - closely tracks the emerging ANSI/ISO C++ standard
  • C++ Compiler supports the "bool/true/false" C++ language extension
  • C++ Compiler supports Apple's Direct-to-SOM language extensions
  • Pascal Compiler - NIST Certified ANS Pascal with MPW Pascal 3.4.1 compatible extensions
  • PowerPC Assembler - Apple's own assembler available as a plug-in compiler
  • CFM68K compiler and linker (builds ODF, OpenDoc® parts, and PowerPlant shared libraries)
  • Balloon help compiler
  • MIPS compiler (beta release for the MIPS processor family)

Features

  • DR1 release of Gamewerks, Metrowerks' cross-platform (Mac and Windows) game development library
  • Full Java support in the IDE - project manager, linker, editor
  • Class-browser for C++, Java, and Object Pascal
  • Metrowerks Java runtime environment (bytecode interpreter) - run applets, droplets and Java applications
  • Java bytecode disassembler - found in Metrowerks Java
  • Applet viewer - view and test applets under development
  • Drag-and-Drop editing in IDE source code editor and preference panels
  • Disassembler in IDE - displays source and assembly simultaneously
  • Error management in IDE - edit errors directly in the error message window
  • Improved IDE - graphical browser view, global versus project preferences, improved external editor
  • Online, interactive help and tutorials in Apple Guide format for the IDE
  • Support for infinite levels of multiple undo
  • C++ templates & zero runtime overhead exception handling
  • RTTI support
  • Global optimization for PowerPC C/C++ compilers
  • Global optimization for Win32/x86 compilers
  • Instruction scheduling for PowerPC 601, 603, and 604 processors, as well as MIPS processors
  • Class hierarchy optimization for Pascal
  • Pointer-based objects in Pascal - objects can be allocated in handles or pointers
  • Procedural types in UPIs for Pascal
  • PowerPlant, a modern mix-in framework, fully supporting the latest Apple technology, now with classes for Internet (ftp, smtp, pop3) / networking development for both client and server.
  • PowerPlant Constructor - visual resource editor for PowerPlant, allows users to visually edit views, menus, menu bars, display class windows and custom pane libraries, now supporting Java
  • Updated support for OpenDoc development for 68K and Power Macintosh
  • MacApp® version 3.3.2 and MacApp2PPC 2.0.1 for Object Pascal
  • Microsoft Foundation Classes®
  • MPW version 3.4.1
  • Metrowerks Standard Library (MSL) for both C and C++, includes source code
  • First release of PowerTV tools - debugger, libraries and samples
  • Final release of General Magic Tools

Debugging

  • Java source-level debugging for Macintosh and Power Macintosh - debug Java code and view both source and assembly (Java bytecodes)
  • Source-level debugger for Macintosh and Power Macintosh with support for SOM objects
  • Mac-hosted, source-level debugger for Magic Cap - Telebug Interface Box and serial cable debugging
  • Mac-hosted, source-level debugger for Win32/x86
  • Mac-hosted, source-level debugger for Be with improved cross-debugger
  • Conditional breakpoints, data watchpoints and expression evaluation
  • Raw memory hex dump in the debugger
  • Debugger for multiprocessor Power Macintosh systems (developer release)

Other Features

  • Two free CD updates included with registration

Minimum Requirements

  • Motorola 68020 processor or higher, or PowerPC 601 processor or higher
  • 8 Megabytes of RAM minimum, 16 Megabytes of RAM recommended (20 Megabytes for Magic Cap development)
  • System 7.1 or later
  • CD-ROM drive for installation
  • 25 Megabytes of free hard disk space

Runs on Macintosh and Power Macintosh.
