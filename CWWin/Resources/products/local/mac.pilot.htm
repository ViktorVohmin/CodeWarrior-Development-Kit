CodeWarrior for Pilot - $299

CodeWarrior® for Pilot offers you the first complete set of development tools to build applications for U.S. Robotics'® new pocket-sized, one-touch organizer, Pilot. Now you can create software programs that will run directly on the Pilot device all from the comfort of your Macintosh® and the award-winning CodeWarrior Integrated Development Environment. Building your own Palm OS-based organizer applications or games is a breeze when using CodeWarrior's intuitive, full-featured IDE with its editor, 68k plug-in compiler and linker, debugger and other familiar tools. Application code compiled in C is linked to a Palm OS library to create a Pilot Simulator application which runs right on the Mac. You can debug and recompile the application from your Macintosh, then download it to the Pilot device for final testing and tuning. It's that easy.

In addition, the CodeWarrior for Pilot toolset includes the Windows®-based Conduit SDK needed to build Palm OS conduit applications. Palm OS conduits are programs that allow Pilot users to synchronize information between the Pilot device and their personal computer by way of the HotSync button found on the Pilot cradle. With CodeWarrior for Pilot you'll find all the Palm OS Conduit SDK for Windows libraries and source files necessary to build conduits to exchange data between your PC applications and Pilot. CodeWarrior for Pilot DR1 users can obtain the Windows SDK directly from Palm Computing.
