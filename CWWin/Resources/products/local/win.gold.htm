CodeWarrior Gold For Windows 95/NT

Windows Hosted Multi-Platform Development Tools

Metrowerks CodeWarrior Gold brings true multi-platform development to Windows NT 4.0 and Windows 95. This complete, industrial-strength programming solution allows you to develop software for Windows NT, Windows 95, Power Macintosh and Macintosh, all using the same Integrated Development Environment (IDE) which Metrowerks has been developing and refining for over two years. CodeWarrior Gold for Windows 95/NT includes C, C++, Pascal and Java compilers that produce fast, highly-optimized code. Complete online documentation, tutorials and sample source code round out the package.

SPECIFICATIONS

Languages Supported:

  • C, C++, Pascal and Java for all target platforms

Target Operating Systems Supported:

  • Mac OS - 68k, PowerPC
  • Windows 95 - x86, Pentium
  • Windows NT 4.0 - x86, Pentium

Features:

  • Disassembler in IDE - displays source and assembly simultaneously
  • Drag and Drop editing in the IDE
  • Error management in the IDE - edit errors directly in the message window
  • C++ templates and zero overhead exception handling
  • Multiple inheritance and RTTI support
  • Class browser for C++, Pascal and Java
  • Java support in the IDE - project manager, editor, linker
  • Java bytecode disassembler
  • Source-level debugger for Win32/x86
  • Conditional breakpoints, data watchpoints and expression evaluation
  • Raw memory hex dump in the debugger
  • Microsoft Foundation Class libraries (MFC 4.2)
  • Microsoft Internet Explorer 3.0
  • Online books
  • Online help and documentation
  • Free technical support
  • 30-day money-back guarantee

Minimum Requirements:

  • 486 processor or higher or Pentium processor or higher
  • 16 MB of RAM
  • Windows 95 or Windows NT 4.0
  • CD-ROM Drive for Installation
