The CodeWarrior IDE

The CodeWarrior IDE (Integrated Development Environment) is the center of CodeWarrior software development. The IDE offers:

  • Power: From the IDE you can edit, browse, compile, link, and run your project. The IDE includes a text editor, class browser, and project manager to help you do all your programming tasks easily and quickly.

  • Flexibility: Plugin compilers and linkers support many different platforms and processors. The same powerful IDE is used to develop and generate software for such targets as Win32(R)/Intel, Mac OS, MIPS(tm), Motorola(R) 680x0, and PowerPC(tm).

  • Convenience: The CodeWarrior IDE for Windows and the CodeWarrior IDE for Mac OS (available in a separate package) are based on the same technology. Versions of the IDE and its plugins for Windows and Mac OS work the same, saving developers time and effort.
